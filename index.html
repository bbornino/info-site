Static Site (AWS S3 + CloudFront)

Overview

This site is a lightweight static site hosted on AWS S3 and delivered through CloudFront. It serves as a sandbox for experimenting with frontend JavaScript and AWS-backed services.

What This Site Demonstrates

  • Static hosting with AWS S3
  • CDN delivery via CloudFront
  • HTTPS using AWS Certificate Manager
  • Automated deployment via AWS CLI
  • Separation from CMS platforms like WordPress
